# ChordPulse Design System

## Direction: Technical Artisan

Enstrüman tamir atölyeleri için teknik, sıcak ve güven veren bir görsel kimlik.

## Color Palette

| Token | Hex | Usage |
|-------|-----|-------|
| Walnut | #3D2B1F | Secondary, sidebar text |
| Copper | #C4783B | Primary accent, wave accent |
| Ivory | #FAF3E8 | Background |
| Espresso | #2A1E16 | Foreground text |

## Typography

- **Display:** JetBrains Mono — başlıklar, marka, teknik his
- **Body:** Nunito Sans — UI metinleri

## Layout

- **Navigation:** Sol sidebar (desktop), drawer (mobil) — FramePulse üst sekmeden farklı
- **Spacing:** 4px taban, kompakt kart padding (16px)
- **Radius:** 2px keskin köşeler

## Components

- `.cp-stat-card` — ince kenarlıklı kartlar
- `.cp-sidebar-link` — sol kenar bakır çizgili aktif menü
- `.cp-wave-accent` — bakır gradient dalga aksanı

## Accessibility

- WCAG AA kontrast
- Light + dark mode
- `aria-label` ve `aria-current` navigasyonda
